
## 开发

项目采用 [react.js](https://reactjs.org/) + [umi.js](https://umijs.org/zh-CN) 开发。

```sh
yarn # 安装模块
npm start # 或者 def dev 启动项目、绑定 host
npm run lint # 检测代码规范

# 本地构建 在 dist 目录启动 server 比如 http://127.0.0.1:3001/
# 拷贝 dist 目录到 warmhug 的 tfox 目录下
npm run build
```
